# Local Build Artifacts

The build scripts place the generated extension packages in the `dist/` folder. Use those files directly for manual testing or local installs.

## Where the files land

- `dist/chromium` for Chromium-based browsers, including Edge
- `dist/firefox` for Firefox
- `dist/safari` for Safari

## How to use them

Follow the manual install guide for the browser you want to test:

- [Microsoft Edge](../getting-started/installation/edge.md)
- [Chrome & Chromium](../getting-started/installation/chrome.md)
- [Firefox](../getting-started/installation/firefox.md)
- [Safari](../getting-started/installation/safari.md)
